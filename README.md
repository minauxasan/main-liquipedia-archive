# main-liquipedia-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容来源于互联网上公开可获取的信息，以静态 HTML 文件的形式存储于此，便于长期保存和访问。

## 仓库说明

- 本仓库不对应任何特定网站或域名，仅作为不同来源 HTML 页面的统一归档点。
- 所有页面均为独立文件，不依赖外部数据库或后端服务。
- 页面内容可能包含原始来源的样式、脚本及资源引用，但仓库本身不保证其完整性或实时更新。
- 归档目的是为研究、学习和历史记录提供参考，不用于任何商业或推广用途。

## 目录结构

```
/
├── pages/          # 存放主要的归档 HTML 页面
├── assets/         # 页面引用的样式、图片、脚本等资源文件（如有）
├── index.html      # 可选的导航或索引页面
└── README.md       # 本说明文件
```

- `pages/` 目录下按时间或主题组织子文件夹，每个子文件夹内包含一个或多个 HTML 文件。
- `assets/` 目录用于存放页面所需的公共资源，避免重复引用。
- `index.html` 仅为方便浏览，非必需。

## 页面归档说明

- 每个 HTML 页面保留其原始结构，不做修改或删减。
- 归档时间以文件提交日期为准，不额外标注原始发布时间。
- 部分页面可能因外部资源链接失效而无法完全正常渲染，仅保留文本内容。
- 仓库不主动更新已归档页面，除非有重大修复需求。

## 维护说明

- 本仓库由维护者不定期更新，添加新的归档页面。
- 不接收来自外部提交的页面归档请求。
- 如果发现页面内容存在明显错误或违规信息，请通过 Issues 提出，维护者将酌情处理。
- 仓库内容遵循开源许可（详见 LICENSE 文件），但页面原始内容可能受第三方版权保护。
